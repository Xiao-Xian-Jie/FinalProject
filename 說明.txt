第一次執行
git branch 自己的代號
git checkout 自己的代號
git add .
git commit -m "自己的代號"
git push --set-upstream origin 自己的代號

需要更新時
先將不是自己的東西砍掉
git checkout 自己的代號
git pull origin master --allow-unrelated-histories
git add .
git commit -m "自己的代號"
git push

寫完程式時
git add .
git commit -m "自己的代號"
git push

不小心把整個或是git的資料夾刪掉
git branch 自己的代號
git checkout 自己的代號
git pull origin 自己的代號 --allow-unrelated-histories
(若出現奇怪的畫面，按 CTRL+Z 或 CTRL+C)
git add .
git commit -m "自己的代號"
git push --set-upstream origin 自己的代號
